## 2. 产品一句话

厂区 **到离场安全准入** 数字化：培训 → 证件 OCR → 预约时段 → 门岗 Check In → 作业/离场检查 → **双签 Check Out** → 电子归档；危货场景叠加 **资质与上海联网联控准入核验**（演示为 Mock，生产走对接）。

---

## 3. 能力边界（当前演示）

### 3.1 已具备

- 四种业务类型：运输入场 / 运输出场 / 客户自提 / 临时车辆  
- 准入三灯（+ 危货第四灯）与硬拦截  
- 时段容量、风险分、通行码、LPR 模拟  
- 入场例外双签、离场司机+门岗双签  
- 中 / EN / DE 界面；Pages 无后端 Mock  
- 危货核验演示：通行码 `HAZ001`（通过）/ `HAZBAD`（目录未核拦截）

### 3.2 明确不做（一期）

- 真实海康/大华 SDK、真实 OCR 云（仅 Adapter 预留）  
- 厂区直连全国重点营运车辆联网联控 **轨迹**平台  
- 多租户 SSO、短信/企微生产通道（stub 可扩展）

## 4. 研发与实施路径（摘要）

| 档位 | 目标 | 增量人天（约） | 详见 |
|------|------|----------------|------|
| A 试点加固 | 单厂试跑；目录人工写回 | ≈ 78 | [人天评估](./产品研发实施人天评估.md) |

npm install
npm run seed
